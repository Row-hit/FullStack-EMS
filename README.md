# Full Stack Employee Management System (EMS)

A modern full-stack Employee Management System built with the MERN stack, focused on real-world business workflows, authentication, payroll management, attendance tracking, leave management, and role-based access control.

## 🚀 Live Demo

Frontend and backend are fully deployed on Vercel for production-ready access.
Live : https://full-stack-ems-fqp6.vercel.app/

## ✨ Features

* Secure JWT Authentication
* Role-Based Access Control (Admin / Employee)
* Employee Management
* Attendance Tracking
* Leave Request System
* Payroll & Payslip Management
* Email Verification using Nodemailer
* Protected Routes & Middleware
* Responsive Modern UI
* REST API Architecture
* Toast Notifications & Error Handling
* Production Deployment on Vercel

## 🛠️ Tech Stack
